# Personal Portfolio & Digital Resume

A clean, responsive, and interactive digital resume website showcasing my professional summary, technical skill sets, software projects, internships, and educational background. Built entirely with semantic HTML5, modern CSS3 variables, and vanilla JavaScript.

🌐 **Live Demo:** [https://my-resume-blue-omega.vercel.app/](https://my-resume-blue-omega.vercel.app/)

---

## 🚀 Features

- **Responsive Design:** Completely optimized for mobile, tablet, and desktop views.
- **Fixed Navigation Bar:** Smooth scroll anchors allow instant jumping between resume sections (`#summary`, `#skills`, `#projects`, etc.).
- **Dynamic Dark/Light Mode:** Toggleable theme layout that respects and saves user preferences via `localStorage`.
- **Modern UI/UX:** Built using custom CSS variables, sticky blurring transitions, and semantic layout tags.

---

## 🛠️ Tech Stack

- **Frontend Structure:** HTML5
- **Styling:** CSS3 (Flexbox, CSS Custom Properties, Media Queries)
- **Interactivity:** JavaScript (ES6+, DOM Manipulation, LocalStorage API)
